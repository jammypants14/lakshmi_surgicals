GST E-INVOICE AND E-WAY BILL

Overview

Introducing ZestyBeanz's 'E-way Bill and GST E-Invoice' module is a powerful add-on designed to simplify and automate the creation of electronic invoices and e-way bills for businesses registered under the GST Act in India. Integrated seamlessly into your Odoo environment, this module streamlines the entire invoicing workflow, ensuring compliance with the latest GST regulations and making the process of managing e-way bills and Invoice Reference Number (IRN) generation as easy as possible.

Key Features

Two Ways to Generate E-Way Bills: Generate e-way bills either using an IRN for validation or independently when the IRN is unavailable.

E-Way Bill and IRN Details: The invoice form view has dedicated tabs for E-Way Bill Details and IRN Details, enabling organized record-keeping.

MasterGST API Integration: The module integrates with MasterGST to send e-way bill and e-invoice requests to the government GST portal via API.

Credential Management: MasterGST credentials (Client ID, Secret ID, Username, and Password) are securely managed in the Dashboard for both sandbox and production environments.

After Installing the Module.

Navigate to settings menu and select a user then enable the 'Manage E-invoice(India)' group to get the module functionalities.

Working

Navigate to Invoicing > Customers > Invoices.

Create a new Invoice and confirm.

Within the invoice form view, under the Actions dropdown, you will find the following options.

1.Create E-Invoice (GST): Generates an electronic invoice and IRN, ensuring compliance with GST rules.

2.Create E-Way Bill (GST) Using IRN: Creates an e-way bill using the previously generated IRN.

3.Create E-Way Bill (GST): Generates a new e-way bill directly, displaying fields such as e-way bill number, creation date, and validity period.

Method 1: Direct Generation

Configure all the details in "E-Way Bill Details" tab of corresponding invoice.

Select "Create E-Way Bill (GST)" option under server actions.

System generates E-way Bill without requiring IRN

Method 2: Using IRN

Configure all the details in "E-Way Bill Details" tab of corresponding invoice.

Select "Create E-Invoice(GST)" option under server actions.

Generates IRN details and show in IRN Details tab.

Select "Create E-Way Bill(GST) Using IRN" option under server actions.

System generates E-way Bill using IRN.

Technical Process Flow

1.ERP sends request to MasterGST with credentials

2.MasterGST validates and forwards request to government GST portal

3.Government portal processes request and returns E-way Bill details

4.System updates invoice with generated information

GST Invoice Report

Users can find the GST invoice report within the Invoice record in the form view. A dedicated button labeled "Print GST Invoice" allows users to download the GST invoice report directly.
